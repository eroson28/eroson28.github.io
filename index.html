    • Hello!

    I'm Erol Sonmez, a CSE student at the Ohio State University with a minor in City & Regional Planning. This is my digital portfolio. Below you can find a catalog of my past projects. You can use the navigation bar on the top left to find me on socials, contact me, or see my resume.

    Thank you for visiting my website.

    Placeify

    A collaborative music-mapping experiment inspired by r/place, where users shape a shared 20×20 digital canvas one tile at a time. Each tile links to a Spotify track, creating a mosaic of music anyone can contribute to. Add a tile and learn about the concept and build process here.

    COTA Forecasting

    An analytical project exploring Columbus transit patterns using geospatial datasets, demographic layers, and COTA bus system data, visualizing trends as a proof of concept for further study in equitable planning. Full methodology and results here.

    FEH Robot

    The final project for OSU’s FEH sequence, this robot (affectionately dubbed Shark B8) was built to navigate a multipart obstacle course. I helped design, prototype, test, and document our approach, mechanical choices, and control algorithms. You can see the full write-up here.

    Block blast

    A Python + Pygame recreation of the mobile puzzle game Block Blast, built to practice game logic, rendering, and basic animations. This was my first time using Pygame and served as a proof of concept. Check out the code or try it yourself here.
